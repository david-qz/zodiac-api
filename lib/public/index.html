Zodiac

This API uses a list of zodiac signs as its resource. The routes are as follows:
Route Method Description Example
/zodiac GET Returns a list of zodiac signs. List Zodiac Signs
/zodiac/all GET Returns more details about all the zodiac signs. Get Details on All Signs
/zodiac/:id GET Returns more details about the zodiac sign matching the id in the parameter. Get Zodiac Sign 1
/zodiac/search?date=[dateString] GET Returns a zodiac sign associated with the birthdate passed in "date" query parameter. Get Zodiac Sign For March 11th
/horoscope/[sign] GET Gets a horoscope for the given sign Get a horoscope for libra
/pages/zodiac GET Generated pages using this api's information. Zodiac Pages
